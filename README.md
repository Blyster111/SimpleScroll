# SimpleScroll

Custom vertical & horizontal scrollview with object pooling, supporting dynamic item size.

- [x] Mouse Scroll
- [x] Touch Scroll
- [x] Scrollbar
- [x] Adjustable scroll speed
- [x] Fast Forward To Index
- [x] Smoothly To Index
- [x] Insert/Edit/Delete Item In Runtime
